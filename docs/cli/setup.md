# `moltbot setup`

初始化 `~/.clawdbot/moltbot.json` 和智能体工作空间。

相关：
- 入门：[入门](/getting-started)
- 向导：[入门](/start/onboarding)

## 示例

```bash
moltbot setup
moltbot setup --workspace ~/clawd
```

通过设置运行向导：

```bash
moltbot setup --wizard
```

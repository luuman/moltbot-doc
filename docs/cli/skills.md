# `moltbot skills`

检查技能（捆绑 + 工作空间 + 管理的覆盖）并查看哪些符合条件与哪些缺少要求。

相关：
- 技能系统：[技能](/tools/skills)
- 技能配置：[技能配置](/tools/skills-config)
- ClawdHub 安装：[ClawdHub](/tools/clawdhub)

## 命令

```bash
moltbot skills list
moltbot skills list --eligible
moltbot skills info <name>
moltbot skills check
```

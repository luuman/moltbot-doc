# AGENTS.md - Moltbot工作空间

此文件夹是助手的工作目录。

## 首次运行（一次性）
- 如果BOOTSTRAP.md存在，请遵循其仪式并在完成后删除它。
- 您的代理身份存在于IDENTITY.md中。
- 您的配置文件存在于USER.md中。

## 备份提示（推荐）
如果您将此工作空间视为代理的"记忆"，请将其设为git仓库（理想情况下是私有的），以便身份
和笔记得到备份。

```bash
git init
git add AGENTS.md
git commit -m "添加代理工作空间"
```

## 安全默认值
- 不要窃取机密或私人数据。
- 除非明确要求，否则不要运行破坏性命令。
- 在聊天中简洁；将较长的输出写入此工作空间中的文件。

## 每日记忆（推荐）
- 在memory/YYYY-MM-DD.md保留简短的每日日志（如需创建memory/）。
- 会话开始时，读取今天+昨天（如果存在）。
- 记录持久事实、偏好和决定；避免机密。

## 心跳（可选）
- HEARTBEAT.md可以包含心跳运行的小清单；保持小巧。

## 自定义
- 在这里添加您喜欢的风格、规则和"记忆"。

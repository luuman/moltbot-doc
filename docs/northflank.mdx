---
title: 在 Northflank 上部署
---

使用一键模板在 Northflank 上部署 Moltbot，并在浏览器中完成设置。
这是最简单的“服务器上无终端”路径：Northflank 为您运行网关，
您通过 `/setup` 网页向导配置一切。

## 如何开始

1. 点击 [部署 Moltbot](https://northflank.com/stacks/deploy-moltbot) 打开模板。  
2. 如果您还没有账户，请在 [Northflank 上创建账户](https://app.northflank.com/signup)。  
3. 点击 **立即部署 Moltbot**。  
4. 设置必需的环境变量：`SETUP_PASSWORD`。
5. 点击 **部署堆栈** 以构建和运行 Moltbot 模板。
6. 等待部署完成，然后点击 **查看资源**。
7. 打开 Moltbot 服务。
8. 打开公共 Moltbot URL 并在 `/setup` 完成设置。
9. 在 `/moltbot` 打开控制 UI。

## 您得到的

- 托管的 Moltbot 网关 + 控制 UI
- `/setup` 的网页设置向导（无终端命令）
- 通过 Northflank 卷（`/data`）的持久存储，因此配置/凭证/工作区在重新部署后依然存在

## 设置流程

1) 访问 `https://<your-northflank-domain>/setup` 并输入您的 `SETUP_PASSWORD`。
2) 选择模型/认证提供者并粘贴您的密钥。
3) （可选）添加 Telegram/Discord/Slack 令牌。
4) 点击 **运行设置**。
5) 在 `https://<your-northflank-domain>/moltbot` 打开控制 UI

如果 Telegram 私信设置为配对，设置向导可以批准配对码。

## 获取聊天令牌

### Telegram 机器人令牌

1) 在 Telegram 中给 `@BotFather` 发消息
2) 运行 `/newbot`
3) 复制令牌（看起来像 `123456789:AA...`）
4) 粘贴到 `/setup` 中

### Discord 机器人令牌

1) 前往 https://discord.com/developers/applications
2) **新建应用程序** → 选择名称
3) **机器人** → **添加机器人**
4) 在机器人 → 特权网关意图下启用 **启用消息内容意图**（必需，否则机器人将在启动时崩溃）
5) 复制 **机器人令牌** 并粘贴到 `/setup`
6) 邀请机器人到您的服务器（OAuth2 URL 生成器；范围：`bot`，`applications.commands`）
